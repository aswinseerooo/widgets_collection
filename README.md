# widgets_collection

This project has been refactored to demonstrate a **clean architecture** approach using:

- `auto_route` for navigation
- `flutter_bloc` for state management
- Domain/data/presentation layers for a simple counter feature

## Getting started

1. Run `flutter pub get` to fetch dependencies.
2. Generate routing code:
   ```bash
   flutter pub run build_runner build --delete-conflicting-outputs
   ```
3. Launch the app (`flutter run`).

The home screen includes a button that navigates to a counter page. The counter value and logic live in the `features/counter` modules following clean architecture principles.

A new Flutter project.
